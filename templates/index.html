Background



The loan application is used by borrowers to apply for a loan. The loan application plays an important role in deciding whether the borrower/user should be granted loan or not. The lender procures basic details from the user via this application which is then further analysed for loan grant.

Problem Statement

The director of SZE bank identified that going through the loan applications to filter the people who can be granted loans or need to be rejected is a tedious and time-consuming process. He wants to automate it and increase his bankâ€™s efficiency.

Objective

The idea behind this ML project is to perform Exploratory Data Analysis, Visualization and build an ML model along with a web application that the bank can use to classify if a user can be granted a loan or not.

Dataset [source]

The dataset contains information about Loan Applicants. There are 12 independent features and 1 target/dependent column. This dataset includes attributes like Loan_ID, Gender, Applicant_Income, if the loan applicant is married or not, the level of education, etc.
